# Task 8: User Authentication

## Overview

Implemented user authentication using NextAuth with Google and credentials. Integrated signup and signin pages with client-side and server-side protection.

## Technologies

- **Next.js**
- **TypeScript**
- **NextAuth.js**
- **React Hook Form**
- **RTK Query & RTK**
